## 기술 및 버전 정보

- **언어:** Java 21
- **프레임워크:** Spring Boot
- **빌드 도구:** Gradle
- **데이터베이스:** MySQL
- **ORM:** JPA
- **캐시/메시지 큐:** Redis
- **문서화:** Swagger (SpringDoc OpenAPI)

## 주요 기술 스택 및 설명

| 기술              | 설명                                |
|-----------------|-----------------------------------|
| **Spring Boot** | 빠른 서버 개발을 위한 프레임워크                |
| **JPA**         | ORM 기반 DB 연동                      |
| **Redis**       | 인메모리 데이터 저장소, 메시지 큐 및 분산락 용도로 활용  |
| **Gradle**      | 빌드 및 의존성 관리 도구                    |
| **MySQL**       | 관계형 데이터베이스                        |
